About Me

At the beginning of this last summer I got my first job within my field. I was hired as a computational research assistant for the Center for Genome Research and Biocomputing. Over the course of the summer I worked on implementing a new online teaching platform for the deparment to make it easier for the biology students to learn how to code and use linux for their data analysis. I structured a system based around docker containers that host both a Jupyter server and an RStudio server for each class in order to isolate each class from over stepping their resource allotments.
